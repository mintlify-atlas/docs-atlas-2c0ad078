title: Configuration reference
description: Environment variables and .env file reference for the GEMS LMS Docker Compose stack.
---

All GEMS LMS services are configured exclusively through environment variables. Docker Compose reads these from a `.env` file at the project root and passes them to each container.

<Warning>
The default database credentials listed below are suitable for local development only. Always set strong, unique passwords before deploying to any shared or production environment.
</Warning>

## Sample .env file

Create this file at the project root before running `docker compose up`.

```bash .env
# API Gateway
API_GATEWAY_PORT=8080

# Education service
EDUCATION_PORT=8081
EDUCATION_DB_PORT=5434
EDUCATION_DB_NAME=education_db
EDUCATION_DB_USER=education_user
EDUCATION_DB_PASSWORD=education_password

# Auth service
AUTH_PORT=8082
AUTH_DB_PORT=5433
AUTH_DB_NAME=auth_db
AUTH_DB_USER=auth_user
AUTH_DB_PASSWORD=auth_password

# Admin service
ADMIN_PORT=8083
ADMIN_DB_PORT=5432
ADMIN_DB_NAME=admin_db
ADMIN_DB_USER=admin_user
ADMIN_DB_PASSWORD=admin_password
```

## API gateway

<ParamField path="body.API_GATEWAY_PORT" type="number" required>
  Host port mapped to the API gateway container's internal port `8080`. All external traffic enters the platform through this port.
</ParamField>

## Education service

<ParamField path="body.EDUCATION_PORT" type="number" required>
  Host port mapped to the `ms-education` container's internal port `8081`.
</ParamField>

<ParamField path="body.EDUCATION_R2DBC_URL" type="string" required>
  R2DBC connection URL passed to the Spring Boot application. Docker Compose constructs this value automatically:

  ```
  r2dbc:postgresql://postgres-education:${EDUCATION_DB_PORT}/${EDUCATION_DB_NAME}
  ```

  The hostname `postgres-education` resolves inside the `gems-lms-network` bridge network.
</ParamField>

<ParamField path="body.EDUCATION_R2DBC_USERNAME" type="string" required>
  Database username passed to Spring R2DBC. Mapped from `EDUCATION_DB_USER`.
</ParamField>

<ParamField path="body.EDUCATION_R2DBC_PASSWORD" type="string" required>
  Database password passed to Spring R2DBC. Mapped from `EDUCATION_DB_PASSWORD`.
</ParamField>

<ParamField path="body.EDUCATION_DB_PORT" type="number" default="5434">
  Host port exposed by the `postgres-education` container. Also used in the R2DBC URL.
</ParamField>

<ParamField path="body.EDUCATION_DB_NAME" type="string" default="education_db">
  PostgreSQL database name created on first startup.
</ParamField>

<ParamField path="body.EDUCATION_DB_USER" type="string" default="education_user">
  PostgreSQL superuser for the education database.
</ParamField>

<ParamField path="body.EDUCATION_DB_PASSWORD" type="string" default="education_password">
  Password for `EDUCATION_DB_USER`.
</ParamField>
